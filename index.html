感谢您选择 ShellRush。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请在使用本游戏前仔细阅读本政策。

使用本游戏即表示您同意本隐私政策的内容。如您不同意，请停止使用本游戏。

一我们收集的信息

1.1 您主动提供的信息

  • 账户注册信息：昵称、头像等游戏内资料（如需注册）
  • 客服联系信息：您联系客服时提供的邮箱地址或其他联系方式
  • 用户反馈内容：您通过反馈功能提交的文字或截图

1.2 我们自动收集的信息

  • 设备信息：设备型号、操作系统版本、设备标识符（如广告 ID）
  • 游戏行为数据：游戏进度、关卡数据、得分记录、游戏时长
  • 日志信息：崩溃报告、错误日志、性能数据
  • 网络信息：IP 地址、网络类型（Wi-Fi / 移动数据）

1.3 我们不收集的信息

本游戏不收集您的真实姓名、身份证号、银行卡信息、精确地理位置或通讯录。

二我们如何使用信息

我们使用所收集的信息用于以下目的：

  • 提供、维护和改善游戏功能与用户体验
  • 保存游戏进度，实现云存档功能（如适用）
  • 分析游戏数据，优化关卡设计和平衡性
  • 检测和修复技术故障、防范作弊行为
  • 处理您的客服请求和反馈
  • 向您发送游戏相关的更新通知（可关闭）

三信息的共享与披露

我们不会出售您的个人信息。仅在以下情形下可能共享您的信息：

  • 服务提供商：与协助我们运营游戏的第三方服务商共享（如数据分析、广告平台、云服务），这些服务商仅限于合同约定的目的使用数据
  • 法律要求：依据法律法规规定、司法或行政机关的要求披露
  • 安全保护：在必要时，为保护本游戏、用户或公众的合法权益而披露

3.1 第三方 SDK 说明

本游戏可能集成以下第三方 SDK，各自有其独立隐私政策：

  • 广告 SDK（如 AdMob、Unity Ads 等）：用于展示广告，可能收集广告标识符
  • 数据分析 SDK（如 Firebase Analytics 等）：用于统计游戏数据
  • 崩溃分析 SDK（如 Crashlytics 等）：用于收集崩溃报告

建议您查阅相关第三方的隐私政策以了解其数据处理方式。

四数据存储与安全

我们采取行业标准的技术措施（包括加密传输、访问控制等）保护您的信息安全。游戏数据主要存储于您的设备本地，如涉及云存档，数据将存储于安全的服务器中。

我们将在实现本政策目的所必要的期限内保留您的数据。您卸载本游戏后，本地数据将随之删除；服务器端数据将在合理期限内按照法律要求予以清除或匿名化。

五儿童隐私保护

ShellRush 不针对 13 周岁以下（或相关法律规定的适用年龄以下）的儿童。我们不会故意收集儿童的个人信息。如果您发现我们无意中收集了未成年人的信息，请联系我们，我们将尽快予以删除。

如您是未成年人，请在监护人的指导下使用本游戏，并请监护人阅读并同意本隐私政策。

六您的权利

根据适用法律，您对您的个人信息享有以下权利：

  • 访问权：了解我们持有的您的个人信息
  • 更正权：要求更正不准确的个人信息
  • 删除权：要求删除您的个人信息（在法律允许的范围内）
  • 撤回同意权：撤回您此前提供的同意（不影响撤回前的处理活动）
  • 限制处理权：要求限制我们对您数据的处理

如需行使上述权利，请通过本政策末尾的联系方式与我们联系。我们将在合理时间内予以响应。

七广告与个性化

本游戏可能展示广告。我们或广告合作伙伴可能使用您的广告标识符（如 IDFA、GAID）提供个性化广告。您可以在设备系统设置中选择「限制广告追踪」或重置广告 ID 来限制个性化广告。

八本政策的变更

我们可能会不时更新本隐私政策。重大变更将通过游戏内通知或其他适当方式告知您。更新后继续使用本游戏，表示您接受更新后的政策。建议您定期查阅本政策。

九联系我们

如对本隐私政策有任何疑问、意见或请求，请通过以下方式联系我们：

  • 电子邮件：privacy@shellrush.com（请替换为您的实际邮箱）
  • 游戏内反馈：设置 → 联系客服
我们将在收到您的请求后 15 个工作日内予以回复。
